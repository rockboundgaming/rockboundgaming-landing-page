# Player of the Month Assets

Place the player's avatar image here and update `potm-data.json` accordingly.

## Default expected filename
`avatar.png`

The `potm-data.json` `avatar` field defaults to `./assets/potm/avatar.png`.
You can use any image format (PNG, JPG, WebP) — just update the path in the JSON.

## Recommended dimensions
- **150 × 150 px** minimum (displayed as a 150px circle)
- Square crop works best
